## Cấu hình AWS CLI

```bash
aws configure
# AWS Access Key ID:     <access key của terraform-admin>
# AWS Secret Access Key: <secret key>
# Default region name:   us-east-1
# Default output format: json
```

Sau đó mở repo hạ tầng để chuẩn bị viết Terraform:

```bash
cd vprofile-infra
code .
```
