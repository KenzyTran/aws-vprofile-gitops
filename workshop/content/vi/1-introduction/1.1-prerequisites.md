## Yêu cầu chuẩn bị

### Tài khoản & dịch vụ

- Tài khoản **AWS** với quyền tạo EC2, EKS, ALB, ACM, ECR, IAM.
- Tài khoản **GitHub** (chứa 3 repo của workshop).
- Một **domain** đã trỏ về AWS và một **public certificate** trong **ACM** (cho HTTPS).

### Công cụ

| Công cụ | Mô tả |
|---------|-------|
| `git` | Quản lý mã nguồn |
| `aws` (AWS CLI) | Tương tác AWS |
| `terraform` | Dựng hạ tầng EKS |
| `kubectl`, `helm` | Thao tác Kubernetes |
| `eksctl` | Tạo IAM service account cho EKS |
| VS Code + AI coding (Copilot / Amazon Q) | Sinh Helm chart, pipeline, Terraform |

{{% notice warning %}}
**Chi phí:** Workshop dùng EKS, EC2 (t2.medium/t3.large) và ALB nên **sẽ phát sinh chi phí**. Hãy
hoàn tất phần Dọn dẹp ở cuối để tránh bị tính tiền ngoài ý muốn.
{{% /notice %}}
